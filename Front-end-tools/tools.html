    # fastmock介绍

    • Mock.js的原理及背景
      • 生成随机数据,拦截Ajax 请求 开始 前后端分离 让前端攻城师独立于后端进行开发。 增加单元测试的真实性 通过随机数据,模拟各种场景
    • postMessage的使用
      • fastmock是一个在线网站，你可以在这个网站配置一个接口，然后你就可以把项目接口的路径改成这个就可以了，这样就可以不用依赖与后端的接口。
